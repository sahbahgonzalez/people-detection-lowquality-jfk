# Data Files

Full datasets are not included in this repository because of GitHub size limits.

## CrowdHuman Dataset
Download from:
https://www.crowdhuman.org/download.html

Extract structure:
crowdhuman/
    images/
    annotation/
    ...

## JFK Frames
These frames originate from a restored Zapruder/JFK dataset.
They are not included in this repository for copyright and size reasons.

The project expects frames in:
data/jfk_frames/

## ESRGAN Models
Real-ESRGAN weights can be downloaded from:
https://github.com/xinntao/Real-ESRGAN/

Place them in:
task1_preprocessing/models/
